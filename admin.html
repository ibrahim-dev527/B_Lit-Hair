📸 Image Storage Setup One-time setup

Run this SQL once in your Supabase SQL Editor to enable cloud image storage. Without it, images are saved as compressed data directly in the database — which still works but is slower for large images.

insert into storage.buckets (id, name, public) values ('product-images', 'product-images', true); create policy "Public can upload images" on storage.objects for insert with check (bucket_id = 'product-images'); create policy "Public can view images" on storage.objects for select using (bucket_id = 'product-images');
